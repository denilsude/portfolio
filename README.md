##  Tech Highlights (Para Recrutadores)

Este portfólio demonstra domínio técnico em:
- **SSG (Static Site Generation):** Foco em performance e SEO.
- **Arquitetura Modular:** Componentização reutilizável e limpa.
- **Mobile First:** Totalmente responsivo para qualquer dispositivo.
- **Modern Stack:** Uso de TypeScript, Astro e PostCSS.

##  Stack Principal
- **Front-end:** Astro, React, Tailwind CSS.
- **Back-end/Automação:** Node.js, Python, Docker.
- **Integrações:** mTLS, API Sicoob, API Omie.
